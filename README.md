# Bango Parc -- Backend Documentation

Written by: [@putrastyo\_](https://instagram.com/putrastyo_)

## Description

Bango Parc is a venue reservation system that allows users to easily and efficiently book spaces online. The system is designed to manage the reservation process, from venue search and account creation to payment confirmation. With comprehensive administration features, Bango Parc helps venue managers organize schedules, manage customers, and monitor payment status in real time.
